# Django Vue BoilerPlate

## Backend setup
```
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
python manage.py runserver
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
